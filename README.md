# python3-ile-sayisal-fizik
İlgili Kitabın 3. Baskısı Olsa Örnekler Nasıl Olurdu?

# Projenin Olayı
2025-2026 itibarı ile etmekte olduğum Matematik lisans eğitimimde Alan Dışı Seçmeli derslerimden biri olarak Fizikte Bilgisayar Programlama'yı seçmiştim, ilgili dersin [kaynak kitabı](https://www.seckin.com.tr/kitap/fortran-ve-python-ile-sayisal-fizik-46-cozumlu-ornek-68-problem-bekir-karaoglu-s-p-947174755)nda ilk baskıdan(2014) kalma FORTRAN kodları ve ikinci baskıda eklenmiş Python 2 kodları bulunuyor. Her ne kadar Python 2 kodlarında ufak değişiklikler yaparak -bazen *ufak* hatalarla- kodlar çalıştırılabiliyor da olsa Python 3'ün şu an daha yaygın kullanılan Python sürümü olması üzerine kodları Python 3'e uyarlamaya karar verdim. Ufak notlarla kitapta anlatılmaya çalışan şeyleri daha da pekiştirmeye çalıştım. Olur da fark ettiğiniz hatalar olursa, issues kısmından iletip erinmezseniz PR olarak çözümünü yapabilirsiniz. Kitabın PDF'ini yahut herhangi bir dijital kopyasını bariz sebeplerden buraya iliştirmiyorum ama olur da kitabı edinmek isterseniz [Turcademy](https://www.turcademy.com/kitap/fortran-ve-python-ile-sayisal-fizik-46-cozumlu-ornek-68-problem-9789750224713)'de dijital olarak mevcut.

# Kullanım Rehberi
init.py'ı tercihen terminalinizde çalıştırın:
```
python init.py
```
Çalıştırmak istediğiniz kodu tercihen terminalinizde çalıştırın:
```
python ./programlar/px-y.py 
```

**Uyarı**: Kodları olabildiğince gözüme nasıl güzel gözüküyorsa o şekilde yazdım, hepsi işlerini yapıyor olsa da her standarta uygun olmayabilir sizin gözünüze güzel gözükmeyebilir. Bu tamamen subjektif bir repo.
